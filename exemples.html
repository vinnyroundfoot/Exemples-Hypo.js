Utilisation des fonctions de Hypo.js

Hypo.convTx - Conversion de taux

Hypo.convTx = function convTx(taux, pOri, pDest, dec)

  • taux : taux à convertir
  • pOri : Périodicité d'origine (1: annuel, 2 : semestriel, 4: trimestriel, 12: mensuel
  • pDest : Périodicité de destination (1: annuel, 2 : semestriel, 4: trimestriel, 12: mensuel
  • dec : nombre de décimales dans le résultat

Hypo.VPM - Calcul de la mensualité d'un emprunt

Hypo.mensualite = Hypo.VPM = function VPM(K0, n, t, dec)

  • K0 : capital emprunté
  • n : durée (exprimée en année, mois, ... selon la périodicité
  • t : taux de l'emprunt (selon la périodicité)
  • dec : nombre de décimales dans le résultat

Hypo.VA - Calcul de la valeur actuelle d'un emprunt sur base de remboursements constants

Hypo.VA = function VA(m, n, t, dec)

  • m : mensualité (remboursement constant)
  • n : durée (exprimée en année, mois, ... selon la périodicité
  • t : taux de l'emprunt (selon la périodicité)
  • dec : nombre de décimales dans le résultat

Hypo.duree - Calcul de la durée de remboursement en fonction du capital de départ, de la mensualité et du taux

Hypo.duree = function duree(K0, m, t, dec)

  • KO : capital emprunté
  • m : mensualité
  • t : taux de l'emprunt (selon la périodicité)
  • dec : nombre de décimales dans le résultat

Hypo.taux - Détermination du taux d'un emprunt en fonction du capital de départ, de la mensualité et de la durée

Hypo.taux = function taux(K0, n, m, f)

Détermination selon la méthode de Newton

  • KO : capital emprunté
  • m : mensualité
  • n : durée
  • f : frais éventuels (intégré dans le taux final)

Hypo.princPer - Calcul de l'amortissement d'une période

Hypo.princPer = Hypo.amortissementPn = function amortissementPn(K0, n, t, p, dec)

  • KO : capital emprunté
  • n : durée
  • t : taux
  • p : période pour laquelle on recherche l'amortissement
  • dec : nombre de décimales dans le résultat

Hypo.intPer - Calcul de l'intérêt d'une période

Hypo.intPer = Hypo.interetsPn = function interetsPn(K0, n, t, p, dec)

  • KO : capital emprunté
  • n : durée
  • t : taux
  • p : période pour laquelle on recherche l'amortissement
  • dec : nombre de décimales dans le résultat

Hypo.cumulPrinc - Calcul de l'amortissement cumulé de la période p1 à la période p2

Hypo.cumulPrinc = function cumulPrinc(K0, n, t, p1, p2, dec)

  • KO : capital emprunté
  • n : durée
  • t : taux
  • p1 : période de départ
  • p2 : période de fin
  • dec : nombre de décimales dans le résultat

Hypo.cumulInt - Calcul de l'intérêt cumulé de la période p1 à la période p2

Hypo.cumulInt = function cumulInt(K0, n, t, p1, p2, dec)

  • KO : capital emprunté
  • n : durée
  • t : taux
  • p1 : période de départ
  • p2 : période de fin
  • dec : nombre de décimales dans le résultat
