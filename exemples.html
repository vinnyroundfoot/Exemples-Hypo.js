Hypo.convTx - Conversion de taux

Hypo.convTx = function convTx(taux, pOri, pDest, dec)
    - taux : taux à convertir
    - pOri : Périodicité d'origine (1: annuel, 2 : semestriel, 4: trimestriel, 12: mensuel
    - pDest : Périodicité de destination (1: annuel, 2 : semestriel, 4: trimestriel, 12: mensuel
    - dec : nombre de décimales dans le résultat

Hypo.VPM - Calcul de la mensualité d'un emprunt

Hypo.mensualite = Hypo.VPM = function VPM(K0, n, t, dec)
- K0 : capital emprunté
- n : durée (exprimée en année, mois, ... selon la périodicité
- t : taux de l'emprunt (selon la périodicité)
- dec : nombre de décimales dans le résultat
